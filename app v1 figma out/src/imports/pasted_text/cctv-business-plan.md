Create a practical internal business management app for CCTV vendors and technicians.

- CCTV shop owners/vendors
- technicians/employees
- service managers

Main problems to solve:
1. Manual billing confusion
2. Difficulty tracking pending works
3. Poor technician coordination
4. Customer/service history management
5. Site maintenance tracking
6. Expense calculation during installations

Core features for Version 1:

1. Customer Management
- Store customer details
- Phone number
- Address
- Site information
- Installation history
- Service notes

2. Work Management
- Create installation requests
- Track maintenance jobs
- Pending/completed status
- Work deadlines
- Service history

3. Technician Assignment
- Assign technician to work
- Track work progress
- Update work status
- Completion confirmation

4. Expense Calculator
- Add camera quantities
- DVR/NVR pricing
- Wire and accessory costs
- Labor charges
- Auto total calculation

5. Billing System
- Generate bills
- Show component breakdown
- Payment status
- Save bill history

App workflow:
Customer Request
→ Vendor Creates Work
→ Technician Assigned
→ Work Completed
→ Bill Generated
→ History Saved

Design requirements:
- Dark modern UI
- Mobile-first design
- Minimal and professional
- Built for real field technicians
- Fast and easy navigation
- Large buttons
- Simple forms
- Operational/business-focused design

Future roadmap:
Version 2:
- Technician app

- Photo uploads
- Real-time status updates

Version 3:
- Multi-vendor platform

Version 4:

Version 5:

The long-term vision is to create a connected ecosystem for CCTV vendors, technicians, suppliers, and customers across India.
